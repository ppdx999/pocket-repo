# PocketRepo

PocketRepoは、**スマートフォンからGitリポジトリを快適に閲覧・探索するための、モバイルファーストなWebアプリケーション**です。

開発者は外出先で「この実装どうなっていたっけ？」「レビューコメントを返したい」「Claude Codeにこのファイルを読ませたい」と思うことがよくあります。
しかし、GitHubのWeb UIや一般的なファイルブラウザはスマートフォンでのコードリーディングに最適化されているとは言えません。

PocketRepoは、自宅や開発マシン上で動作する軽量なサーバーとして動き、Tailscaleなどのプライベートネットワーク経由でスマートフォンから安全にアクセスします。

## 目標

* スマホでコードを読むことに最適化されたUI
* Gitリポジトリを高速に閲覧できること
* 専用アプリ不要でブラウザから利用できること
* 単一バイナリで簡単に導入できること
* プライベートネットワーク内で安全に利用できること

## 初期機能（MVP）

* 複数リポジトリの管理
* ディレクトリツリー表示
* ファイル閲覧
* シンタックスハイライト
* Git diff表示
* ファイル検索
* 最近閲覧したファイルの表示

## 将来的な機能

* Claude Codeセッションとの連携
* 「このファイルを読んで」「この差分をレビューして」といったAIへの送信
* シンボル検索
* Git blame表示
* Pull Requestレビュー支援
* ターミナル出力の閲覧
* 複数開発マシン・複数リポジトリの管理
* プッシュ通知によるAIタスク完了通知

PocketRepoは「スマホでコードを読むためのツール」から始まり、最終的には**ポケットの中にある開発環境**を目指します。
